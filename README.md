# Game Assistant - 游戏辅助工具

一个基于Python的Windows游戏辅助工具，支持屏幕捕获、图像识别、鼠标键盘模拟和脚本录制回放。

## 功能特性

### 核心功能
- **窗口管理**: 枚举和选择目标游戏窗口
- **屏幕捕获**: 高性能截图（使用mss库）
- **图像识别**: OpenCV模板匹配，支持多目标识别
- **输入模拟**: 使用Win32 API底层模拟鼠标键盘（降低检测风险）

### 脚本录制
- 录制鼠标点击、移动和键盘操作
- JSON格式存储，可编辑
- 支持变速回放和循环播放

### 界面功能
- 现代化GUI界面（PySide6）
- 全局热键支持（F9开始/停止，F10暂停，F12紧急停止）
- 系统托盘最小化

## 安装

### 环境要求
- Windows 10/11
- Python 3.9+

### 安装依赖

```bash
cd game_assistant
pip install -r requirements.txt
```

依赖库列表：
- mss - 屏幕捕获
- opencv-python - 图像处理
- numpy - 数组操作
- pywin32 - Windows API
- pynput - 输入监听
- keyboard - 全局热键
- PySide6 - GUI框架
- Pillow - 图像处理

## 使用方法

### 启动程序

```bash
python main.py
```

### 基本操作流程

1. **选择目标窗口**
   - 在主界面顶部的下拉框中选择目标游戏窗口
   - 点击"刷新"更新窗口列表
   - 点击"预览"查看窗口截图

2. **脚本录制**
   - 切换到"脚本录制"选项卡
   - 点击"开始录制"按钮
   - 执行你想要自动化的操作
   - 点击"停止录制"
   - 点击"保存录制"并输入脚本名称

3. **脚本回放**
   - 在脚本列表中选择要播放的脚本
   - 设置回放速度和循环次数
   - 点击"播放"按钮
   - 使用F10暂停/继续，F12紧急停止

4. **图像识别**
   - 切换到"图像识别"选项卡
   - 添加模板图像（要识别的目标）
   - 设置匹配阈值
   - 勾选"点击匹配位置"自动点击找到的位置
   - 点击"测试识别"验证

### 热键

| 热键 | 功能 |
|------|------|
| F9 | 开始/停止 |
| F10 | 暂停/继续 |
| F12 | 紧急停止 |

热键可在设置选项卡中自定义。

## 注意事项

⚠️ **重要提示**：

1. **游戏兼容性**: 部分游戏可能有反作弊系统，使用此工具可能导致账号被封禁。建议仅用于单机游戏或明确允许自动化的场景。

2. **管理员权限**: 某些游戏需要以管理员权限运行本工具才能正常模拟输入。

3. **法律责任**: 用户需自行承担使用此工具的法律责任。开发者不对任何因使用此工具造成的损失负责。

## 开发扩展

### 添加新的识别方法

在 `core/recognition.py` 中扩展 `ImageRecognition` 类。

### 自定义输入模拟

在 `core/input.py` 中的 `InputSimulator` 类添加新方法。

### 添加新的GUI功能

在 `gui/widgets/` 目录下创建新的控件。
