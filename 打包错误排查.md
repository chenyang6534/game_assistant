# 打包错误排查指南

## 错误：Permission denied (权限被拒绝)

### 错误信息示例
```
PermissionError: [Errno 13] Permission denied: 'D:\\pytest1\\game_assistant\\dist\\GameAssistant\\_internal\\ucrtbase.dll'
```

---

## 解决方法（按顺序尝试）

### 方法 1：关闭正在运行的程序 ⭐⭐⭐⭐⭐
**最常见原因**

1. 按 `Ctrl + Shift + Esc` 打开任务管理器
2. 在"进程"选项卡中查找 `GameAssistant.exe`
3. 如果找到，右键点击 → 结束任务
4. 关闭任务管理器
5. 重新运行 `打包成EXE.bat`

---

### 方法 2：使用清理脚本 ⭐⭐⭐⭐⭐
**推荐方法**

1. 双击运行 `清理打包文件.bat`
2. 等待清理完成
3. 重新运行 `打包成EXE.bat`

如果清理脚本提示删除失败：
- 右键 `清理打包文件.bat` → **以管理员身份运行**

---

### 方法 3：手动删除文件夹 ⭐⭐⭐⭐
**清理脚本失败时使用**

1. 找到这两个文件夹：
   - `game_assistant\dist`
   - `game_assistant\build`

2. 尝试直接删除这两个文件夹

3. 如果删除失败，显示"文件正在使用"：
   - 重启电脑
   - 或使用下面的"方法 6"

---

### 方法 4：以管理员身份运行 ⭐⭐⭐⭐
**权限不足时使用**

1. 右键点击 `打包成EXE.bat`
2. 选择 **"以管理员身份运行"**
3. 在弹出的用户账户控制窗口点击"是"

---

### 方法 5：关闭杀毒软件 ⭐⭐⭐
**杀毒软件干扰**

常见杀毒软件：
- Windows Defender
- 360安全卫士
- 腾讯电脑管家
- 火绒安全

**临时关闭步骤（以 Windows Defender 为例）**：
1. 打开 Windows 安全中心
2. 病毒和威胁防护 → 管理设置
3. 临时关闭"实时保护"
4. 运行打包脚本
5. 完成后记得重新开启

**或添加到白名单**：
1. Windows 安全中心 → 病毒和威胁防护 → 管理设置
2. 排除项 → 添加或删除排除项
3. 添加文件夹：`D:\pytest1\game_assistant`

---

### 方法 6：使用 Unlocker 工具解锁文件 ⭐⭐⭐
**文件被占用无法删除时使用**

1. 下载 Unlocker 工具（免费）
   - 官网：https://www.iobit.com/en/iobit-unlocker.php
   - 或搜索"IObit Unlocker"

2. 安装后，右键 `dist` 文件夹
3. 选择"IObit Unlocker"
4. 点击"解锁"或"删除"

---

### 方法 7：检查磁盘空间 ⭐⭐
**空间不足**

1. 打开"此电脑"
2. 查看 D 盘可用空间
3. 确保至少有 **2GB** 可用空间
4. 如果不足，清理磁盘或更换盘符

---

### 方法 8：更换打包位置 ⭐⭐
**路径问题**

如果路径包含：
- 中文字符
- 特殊符号
- 过长的路径

尝试将项目移动到简单路径，例如：
```
C:\GameAssistant\
```

---

### 方法 9：使用命令行手动打包 ⭐
**脚本失败时的备用方案**

1. 以管理员身份打开 PowerShell
2. 进入项目目录：
   ```powershell
   cd D:\pytest1\game_assistant
   ```

3. 手动删除旧文件：
   ```powershell
   Remove-Item -Recurse -Force dist -ErrorAction SilentlyContinue
   Remove-Item -Recurse -Force build -ErrorAction SilentlyContinue
   Remove-Item *.spec -ErrorAction SilentlyContinue
   ```

4. 运行打包命令：
   ```powershell
   pyinstaller --name=GameAssistant --windowed --onedir main.py
   ```

---

## 其他常见错误

### 错误：找不到 PyInstaller
**解决方法**：
```bash
pip install pyinstaller
```

### 错误：找不到某个模块
**解决方法**：
```bash
pip install -r requirements.txt
```

### 错误：Python 版本过低
**解决方法**：
- 升级到 Python 3.9 或更高版本
- 下载地址：https://www.python.org/downloads/

---

## 预防措施

在打包前：
1. ✅ 关闭所有可能运行的 GameAssistant.exe
2. ✅ 暂时关闭杀毒软件
3. ✅ 确保以管理员身份运行
4. ✅ 确保磁盘空间充足（至少 2GB）
5. ✅ 关闭可能占用文件的程序（如资源管理器预览）

---

## 仍然无法解决？

### 最终方案：重启电脑后重试
1. 保存所有工作
2. 重启电脑
3. 不要运行任何 GameAssistant 相关程序
4. 以管理员身份直接运行 `打包成EXE.bat`
