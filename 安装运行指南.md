# Game Assistant 安装运行指南

本程序可以在任何Windows 10/11电脑上运行，有两种方式：

## 方式一：使用Python运行（推荐开发者使用）

### 前置要求
- Windows 10/11 操作系统
- Python 3.9 或更高版本

### 步骤 1：安装Python
1. 访问 https://www.python.org/downloads/
2. 下载最新的 Python 3.x 版本（建议 3.9+）
3. **安装时务必勾选 "Add Python to PATH"**
4. 点击"Install Now"完成安装

### 步骤 2：验证Python安装
打开命令提示符（按 Win+R，输入 cmd），运行：
```bash
python --version
```
应该显示 Python 版本号，如：Python 3.11.0

### 步骤 3：复制程序文件
将整个 `game_assistant` 文件夹复制到目标电脑的任意位置，例如：
```
C:\Users\用户名\Documents\game_assistant\
```

### 步骤 4：安装依赖库
在命令提示符中，进入程序目录：
```bash
cd C:\Users\用户名\Documents\game_assistant
```

安装所需依赖：
```bash
pip install -r requirements.txt
```

等待所有依赖安装完成（可能需要几分钟）。

### 步骤 5：运行程序
**以管理员权限运行**（某些游戏需要）：
1. 在 `game_assistant` 文件夹中找到 `main.py`
2. 右键点击，选择"以管理员身份运行 PowerShell"
3. 输入：
```bash
python main.py
```

或者创建快捷方式（见下文）。

---

## 方式二：打包成EXE（推荐普通用户使用）

### 适用场景
- 在没有Python环境的电脑上运行
- 需要分发给其他用户
- 不想暴露源代码

### 步骤 1：在开发电脑上安装PyInstaller
```bash
pip install pyinstaller
```

### 步骤 2：打包程序
进入 `game_assistant` 目录，运行：
```bash
pyinstaller --name="GameAssistant" ^
    --windowed ^
    --icon=icon.ico ^
    --add-data="templates;templates" ^
    --add-data="config.json;." ^
    --hidden-import=win32gui ^
    --hidden-import=win32api ^
    --hidden-import=win32con ^
    --hidden-import=pywintypes ^
    main.py
```

如果没有图标文件，去掉 `--icon=icon.ico` 这行。

### 步骤 3：等待打包完成
打包过程可能需要 5-10 分钟。完成后会在 `dist` 文件夹中生成 `GameAssistant` 文件夹。

### 步骤 4：分发程序
将 `dist/GameAssistant` 整个文件夹复制到目标电脑，包含：
- GameAssistant.exe（主程序）
- 所有 .dll 文件
- templates 文件夹（如果有）
- 其他资源文件

### 步骤 5：运行EXE
1. 找到 `GameAssistant.exe`
2. 右键 → "以管理员身份运行"

## 创建桌面快捷方式

### 方式一（源码运行）
1. 在桌面右键 → 新建 → 快捷方式
2. 位置输入：
   ```
   C:\Windows\System32\cmd.exe /c "cd /d C:\Users\用户名\Documents\game_assistant && python main.py"
   ```
3. 名称输入：Game Assistant
4. 右键快捷方式 → 属性 → 高级 → 勾选"用管理员身份运行"

### 方式二（EXE运行）
1. 找到 `GameAssistant.exe`
2. 右键 → 发送到 → 桌面快捷方式
3. 右键快捷方式 → 属性 → 高级 → 勾选"用管理员身份运行"

## 常见问题

### Q1：提示缺少依赖库
**解决方法**：重新运行
```bash
pip install -r requirements.txt
```

### Q2：提示"Python不是内部或外部命令"
**解决方法**：
1. 重新安装Python，勾选"Add Python to PATH"
2. 或手动添加Python到环境变量

### Q3：程序无法模拟输入
**解决方法**：
1. 以管理员权限运行程序
2. 检查目标游戏是否有反作弊保护

### Q4：找不到目标窗口
**解决方法**：
1. 确保游戏窗口已打开
2. 点击"刷新"按钮更新窗口列表
3. 某些全屏游戏需要切换到窗口模式

### Q5：图像识别不准确
**解决方法**：
1. 调整识别阈值（降低阈值提高匹配率）
2. 确保模板图片清晰
3. 使用"预览"功能检查截图质量

### Q6：打包的EXE体积过大
**解决方法**：使用虚拟环境打包可减小体积：
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
pip install pyinstaller
pyinstaller ...（打包命令）
```

### Q7：杀毒软件误报
**解决方法**：
1. 添加到白名单
2. 使用代码签名证书（专业用途）

## 系统要求
- **操作系统**：Windows 10/11（64位）
- **内存**：建议 4GB 以上
- **磁盘空间**：
  - 源码运行：约 500MB（包括Python和依赖）
  - EXE运行：约 200MB
- **权限**：需要管理员权限（用于输入模拟）

---

## 注意事项

1. **安全性**：本程序仅供学习交流使用，请勿用于非法用途
2. **游戏规则**：使用前请确认是否违反游戏服务条款
3. **杀毒软件**：某些杀毒软件可能误报，因为程序使用了底层输入模拟
4. **管理员权限**：建议始终以管理员身份运行，否则可能无法控制某些程序

---

## 更新程序

### EXE方式
重新打包并替换 dist 文件夹中的文件。
